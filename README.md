# Glassmorphic Modal UI

An interactive UI component built as part of my web development journey. This project demonstrates DOM manipulation with JavaScript and modern CSS styling techniques.

## Features
- **Glassmorphism UI:** Translucent backgrounds with frost-blur effects.
- **Smooth Animations:** Custom CSS keyframes for a professional entry/exit feel.
- **Clean Code:** Vanilla JavaScript (no frameworks) for optimal performance.
- **Responsive Design:** Optimized for different screen sizes.

## Technologies Used
- HTML5
- CSS3 (Flexbox, Keyframes, Backdrop-filter)
- JavaScript (DOM Manipulation)
- Google Fonts API (Poppins)
